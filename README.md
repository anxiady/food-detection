# Food Detection

A food detection project using computer vision and machine learning.

## Getting Started

This project is currently in development.

## Requirements

- Python 3.x
- OpenCV
- TensorFlow/PyTorch (depending on the model used)

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
